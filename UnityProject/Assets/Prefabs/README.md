# Prefabs

Most commerce demo objects are generated from primitives at runtime by
`Scripts/SceneObjectFactory.cs`, driven by the `sceneObjectType` field in the
product data.

`HouseModel_ScannedPlan.prefab` is a baked procedural model generated by
`Scripts/House/HouseModelGenerator.cs` for the scanned residential plan set.

When real assets arrive (glTF models, Addressables), replace factory builders
with prefab lookups here — the data contract stays the same.
