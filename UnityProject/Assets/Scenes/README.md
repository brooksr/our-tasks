# Scenes

The commerce demo builds its object graph at runtime (`Scripts/AppBootstrap.cs`),
so `Main.unity` is intentionally empty.

Create the (empty) build scene with **Tools → Commerce Demo → Create Main Scene**
in the Unity editor — the build script also creates it automatically if missing.

`HouseModel_ScannedPlan.unity` is a standalone generated inspection scene for
the scanned residential house model.
